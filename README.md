# Seva Sarthi Associates - Official Website

A modern, professional, and fully responsive Single Page Application (SPA) for **Seva Sarthi Associates**, a premier real estate and legal advisory firm based in Delhi NCR.

## 📖 Project Overview

This website serves as the digital presence for Seva Sarthi Associates, showcasing their expertise in commercial real estate, industrial plots, interior designing, and legal advisory services. The design is tailored to reflect trust, transparency, and professionalism with a corporate White & Blue theme.

## ✨ Features

- **Responsive Design**: Seamlessly adapts to Mobile, Tablet, and Desktop screens.
- **Single Page Navigation**: Smooth scrolling across all sections (Home, About, Services, Founder, Contact).
- **Interactive Elements**: Mobile hamburger menu, hover effects on cards, and clean transitions.
- **Modern UI**: Professional typography (Inter & Playfair Display), grid layouts, and aesthetic spacing.
- **Contact Integration**: Easy access to phone numbers, email, and office addresses.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure and content.
- **CSS3**: Custom styling, Flexbox/Grid layouts, and CSS variables for theming.
- **JavaScript (Vanilla)**: Mobile menu toggling and smooth scrolling logic.
- **Google Fonts**: 'Inter' for body text and 'Playfair Display' for accents.

## 🚀 Setup & Usage

Since this is a static website, no complex installation or build process is required.

1. **Clone or Download** the project folder.
2. **Open** the `index.html` file in any modern web browser (Chrome, Firefox, Edge, Safari).
3. **Explore**: Navigate through the menu to view different sections.

### Folder Structure

```
/Seva sarthi
│
├── index.html          # Main HTML file
├── README.md           # Project documentation
│
├── css/
│   └── style.css       # Global styles and responsive rules
│
├── js/
│   └── main.js         # Navigation script
│
└── assets/
    ├── images/         # Project images (Logo, Founder, Hero BG)
    └── icons/          # Service icons
```

## 🎞️ Customization

- **Images**: Place your actual images in the `assets/images` folder.
  - Logo: `logo.png`
  - Hero Background: `hero_bg.png`
  - Founder Image: `vinod.jpeg` (Updated in code)
- **Text**: Edit `index.html` to update content, phone numbers, or addresses.
- **Colors**: Modify the `:root` variables in `css/style.css` to change the color scheme.
